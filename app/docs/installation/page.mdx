# Installation Guide

## Prerequisites

- **Node.js** v18+
- **React** v18+
- **shadcn/ui** installed

## Installation Options

You can install **prompt-kit** components using either the **prompt-kit CLI** or the **shadcn CLI**.

### Using the shadcn CLI:

```bash
npx shadcn@latest add prompt-kit/{component}
```

### Usage

After installation, import and start using the components in your project:

```tsx
import { PromptInput } from '@/components/ui/prompt-input';
```
