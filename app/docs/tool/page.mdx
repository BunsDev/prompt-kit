## Component API

### Tool

| Prop           | Type           | Default | Description                                      |
| :------------- | :------------- | :------ | :----------------------------------------------- |
| toolInvocation | ToolInvocation |         | The tool invocation data to display              |
| defaultOpen    | boolean        | false   | Whether the tool details are expanded by default |
| className      | string         |         | Additional CSS classes                           |

### ToolGroup

| Prop        | Type             | Default | Description                              |
| :---------- | :--------------- | :------ | :--------------------------------------- |
| tools       | ToolInvocation[] |         | Array of tool invocations to display     |
| defaultOpen | boolean          | false   | Whether the group is expanded by default |
| className   | string           |         | Additional CSS classes                   |
| title       | string           | "Tools" | Title to display for the group           |

### ToolInvocation Type

| Property | Type                                             | Description                       |
| :------- | :----------------------------------------------- | :-------------------------------- |
| toolName | string                                           | Name of the tool                  |
| state    | "pending" \| "running" \| "completed" \| "error" | Current execution state           |
| args     | `Record<string, unknown>`                        | Arguments passed to the tool      |
| result   | unknown                                          | Result returned by the tool       |
| error    | string                                           | Error message if execution failed |
